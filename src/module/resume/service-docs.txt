**ResumeService** — CRUD для резюме, 5 методов:

**`createResume`** — создаёт резюме из DTO и сохраняет в БД

**`getAllResume`** — возвращает все резюме без фильтров

**`getResumeById`** — ищет резюме по `id`

resume_response  - ./resume_response.service.ts  => для отклика резюме от рабодатателя 

**`removeResume`** — удаляет резюме
1. Сначала ищет — нет = 404
2. Удаляет и возвращает удалённый объект

**`updateResume`** — обновляет резюме
1. Сначала ищет — нет = 404
2. `Object.assign` мержит новые данные поверх старых
3. Сохраняет
